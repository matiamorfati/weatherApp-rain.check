The main goal was to practice working with real-world APIs, asynchronous JavaScript, and dynamic DOM manipulation.

Through this project, I learned how to fetch and handle data from an external weather API, process and structure that data,
and render it dynamically on the page. I also improved my understanding of JavaScript modules, separation of concerns (logic vs. UI),
and organizing code into reusable components. Additionally, I focused on creating a responsive and user-friendly interface,
handling user input, and managing application state.
